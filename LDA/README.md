# LDA
- 토픽모델링으로 현재 단계에서 여행지 설명글에 토픽은 1개라고 가정하고 토픽을 구성하는 단어의 갯수는 20개만을 추출하여 거리를 구하여 보았다.
- 토픽의 갯수를 늘려보는 방식으로 해당 모델을 튜닝 할 수 있을것으로 판단됨.
